# Projet de Gestion de Plateforme Web - ENEAM / UAC

Ce projet est une application web dynamique permettant la gestion des articles, des clients et des ventes. Il a été développé dans le cadre de mon cursus académique.

## 🚀 Fonctionnalités
- Gestion des utilisateurs (Inscription / Connexion).
- Gestion du catalogue d'articles.
- Suivi des clients et enregistrement des ventes.
- Interface d'administration sécurisée.

## 🔐 Accès Administration
Pour tester les fonctionnalités d'administration, utilisez les identifiants suivants :
- *Utilisateur :* admin
- *Mot de passe :* admin123

## 🛠️ Installation et Configuration locale
Pour faire fonctionner ce projet sur votre machine (via XAMPP/WAMP) :

1. *Base de données :*
   - Créez une base de données nommée projet_web dans votre outil de gestion (phpMyAdmin).
   - Importez le fichier projet_web.sql présent à la racine de ce dépôt.

2. *Fichiers :*
   - Placez l'intégralité des fichiers du dépôt dans votre dossier serveur (ex: C:/xampp/htdocs/projet_web/).

3. *Connexion :*
   - Vérifiez le fichier connexion.php ou myparam.inc.php pour vous assurer que les paramètres de connexion correspondent à votre configuration locale.

4. *Lancement :*
   - Accédez au site via http://localhost/projet_web/index.php.

## 📁 Structure du projet
- /images : Contient les logos (ENEAM, UAC) et les ressources graphiques.
- *.php : Scripts de traitement et pages du site.
- style.css : Feuille de style pour le design.
- projet_web.sql : Script d'export de la base de données.
